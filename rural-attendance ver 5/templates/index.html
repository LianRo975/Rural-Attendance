About the Application

  • Multi-school support; classes LKG–12; sections; roles: Admin, Teacher, Student.
  • CRUD: Schools, Classes, Sections, Teachers, Students, Subjects, Periods, Teacher Assignments.
  • Period-wise attendance: per class × section × period × date; quick mark all; save anytime.
  • Daily class report: end-of-day summary by section (Present / Absent / Total).
  • Fast seeder: optional purge + bulk seed of schools/teachers/subjects/periods/students + 1–2 week timetable.
  • Role-based dashboards & navigation; audit-friendly timestamps on all entities.
  • Tech stack: HTML/JS (frontend), Flask (backend), SQLite (DB). Face recognition & RFID integration next.
  • Responsive, lightweight UI; edge-to-edge layout for wide screens.

Note: Face/vision capture screens will be added in upcoming versions.
